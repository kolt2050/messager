# Messenger

Чат-приложение в реальном времени, созданное с использованием современного React фронтенда и надежного FastAPI бэкенда.

## Обзор

Этот проект представляет собой полнофункциональное чат-приложение, позволяющее пользователям общаться в режиме реального времени. Оно включает в себя безопасную систему аутентификации, постоянное хранение сообщений и отзывчивый пользовательский интерфейс.

## Технологический стек

### Фронтенд (Клиент)
- **Фреймворк**: [React](https://react.dev/)
- **Сборщик**: [Vite](https://vitejs.dev/)
- **Стилизация**: CSS Modules / Standard CSS
- **Маршрутизация**: React Router
- **HTTP Клиент**: Axios

### Бэкенд (Сервер)
- **Фреймворк**: [FastAPI](https://fastapi.tiangolo.com/)
- **ORM Базы Данных**: [SQLAlchemy](https://www.sqlalchemy.org/)
- **Аутентификация**: JWT (JSON Web Tokens) с использованием `python-jose` и `passlib`
- **Связь в реальном времени**: WebSockets

## Предварительные требования

Перед началом работы убедитесь, что у вас установлено следующее:
- **Node.js** (версия 18 или выше)
- **Python** (версия 3.10 или выше)
- **Docker** и **Docker Compose** (Опционально, для развертывания в контейнерах)

## Установка и Запуск

Вы можете запустить приложение либо с помощью Docker (рекомендуется для простоты), либо вручную.

### Вариант 1: Docker (Рекомендуется)

1. **Клонируйте репозиторий** (если вы еще этого не сделали):
   ```bash
   git clone <url-репозитория>
   cd messager
   ```

2. **Запустите приложение**:
   ```bash
   docker-compose up --build
   ```
   Эта команда соберет образы для клиента и сервера и запустит контейнеры.

   **Запуск по отдельности:**
   - Только сервер: `docker-compose -f docker-compose.server.yml up --build`
   - Только клиент: `docker-compose -f docker-compose.client.yml up --build`

3. **Доступ к приложению**:
   - Фронтенд: `http://localhost:5173` (или порт, указанный в docker-compose)
   - Документация API Бэкенда: `http://localhost:8000/docs`

### Вариант 2: Ручная Установка

#### Бэкенд (Сервер)

1. Перейдите в директорию сервера:
   ```bash
   cd server
   ```

2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/bin/activate  # На Windows: venv\Scripts\activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите сервер:
   ```bash
   uvicorn main:app --reload
   ```
   Сервер запустится по адресу `http://127.0.0.1:8000`.

#### Фронтенд (Клиент)

1. Перейдите в директорию клиента:
   ```bash
   cd client
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
   Клиент будет доступен по адресу `http://localhost:5173`.
